# API Reference

This section provides detailed API documentation for all SPXQuery modules.

The package is organized into four main module groups:

## Core Modules

Pipeline orchestration, data acquisition, and configuration.

- {doc}`core` - Query execution, downloads, pipeline management, and configuration classes

## Processing Modules

Scientific data extraction and analysis.

- {doc}`processing` - FITS handling, photometry, light curves, and magnitude conversions

## Visualization Modules

Publication-quality plotting with quality control.

- {doc}`visualization` - Combined spectral and temporal plots

## Utility Modules

Shared utilities and parameter management.

- {doc}`utils` - Logging, JSON I/O, parameter export/import
